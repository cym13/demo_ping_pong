# PingPong - Simple crypto CTF

## Concept

Without looking at the source code, run and explore the small website. You'll
find that its URLs are encrypted. This poses 3 challenges:

- Decrypt all URLs
- Gain access to the administrator's secret
- Dump the list of all users

*Be warned: if you look at the source code you will be spoiled.*

## Starting the website

This project uses uv, so you should be able to run it simply with:

```
$ uv run main.py
```
